# Hearth Desktop Sprint Task

## Goal
Build ONE Tauri feature or Svelte component:
- System tray integration, notifications, or UI enhancement

## Constraints
- Focus on a single, complete feature
- Must compile and run
- Clean code, proper error handling

## Steps
1. Review current state of the codebase
2. Identify missing feature (system tray, notifications, or UI)
3. Implement the feature
4. Test locally
5. Commit and push

Start by examining the project structure and current features.
